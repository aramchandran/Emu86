About Emu86

This is an open source project written by Gene Callahan, Adnan Bin Hoq, and Syed Maudud E Rabbi with the aid of Paul Logston. The goal is to create a simple learning tool to introduce students to assembly language programming in a class on, say, operating systems or compilers, where the focus is not assembly language, and actually using an assembler would be a distraction from the main theme of the course. It includes a (growing) subset of x86 assembler instructions. (This is, by the way, Intel assembly language, and not AT&T assembly language.)

Instructions Available

Many of these are only partially done: for instance, as of yet, there is no memory addressing available.
In the syntax descriptions, we have:

  • reg: register
  • con: constant
  • MOV
    Syntax available:
    • MOV reg, reg
    • MOV reg, con
  • ADD
    Syntax available:
    • ADD reg, reg
    • ADD reg, con
  • SUB
    Syntax available:
    • SUB reg, reg
    • SUB reg, con
  • IMUL
    Syntax available:
    • IMUL reg, reg
    • IMUL reg, con
  • AND
    Syntax available:
    • AND reg, reg
    • AND reg, con
  • OR
    Syntax available:
    • OR reg, reg
    • OR reg, con
  • XOR
    Syntax available:
    • XOR reg, reg
    • XOR reg, con
  • SHL
    Syntax available:
    • SHL reg, reg
    • SHL reg, con
  • SHR
    Syntax available:
    • SHR reg, reg
    • SHR reg, con
  • NOT
    Syntax available:
    • NOT reg
